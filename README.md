# Apartment-Prediction
> 부동산 데이터를 활용한 아파트 실거래가 예측

## 프로젝트 요약
* 기간 : 2022.01 ~ 2022.02
* 주관 : 서울시립대학교 스마트시티교육연구단
* 데이터 출처 : 데이콘, 서울열린데이터광장
* 역할 : 데이터 수집, EDA, 모델링(LightGBM) 
* 사용한 모델 : Random Forest, XGBoost, LightGBM, CatBoost
* 시각화
